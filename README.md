# IntelliZen

macOS-only Tauri v2 desktop intelligence platform for GenZen.

Current V1 scope:

- Inbox
- Search
- Projects
- Monitors
- Graph

Stack:

- Tauri v2
- React 18 + TypeScript + Vite
- Tailwind CSS v4
- Zustand
- TanStack Query
- Supabase JS v2
- Exa JavaScript SDK (`exa-js`)
- React Flow
